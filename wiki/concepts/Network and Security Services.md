# Network and Security Services

## Definition

Network & Security is one of the four core services [[CK Technology LLC]] markets
on [[cktechx.com]]: enterprise-grade network design plus layered cybersecurity. The
stated philosophy is "security as a foundation, not an afterthought" — built into
everything from network design to endpoint protection to user training.

## What the Service Includes (public marketing)

- **Firewall & network security** — enterprise firewalls, VPN, network segmentation.
- **Endpoint protection** — EDR (endpoint detection and response) for workstations
  and servers.
- **Email security** — spam filtering, phishing protection, email encryption.
- **Security monitoring** — continuous monitoring with alerting and incident response.
- **Network design** — Wi-Fi, switching, and connectivity infrastructure.
- **Security assessments** — vulnerability assessments and security reviews.

## Positioning: Defense in Depth

The page frames the offering around **defense in depth** — layered controls so no
single point of failure can compromise the business, protecting from the network edge
down to individual endpoints.

> [!key-insight]
> "Security first" is also one of the company's three core values (see
> [[MSP Positioning and Values]]), so this service is the natural commercial
> expression of that value rather than a bolt-on. It maps cleanly onto the founder's
> own infrastructure background showcased on [[ckelley.dev]].
