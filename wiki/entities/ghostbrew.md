# ghostbrew

An Arch Linux package management/discovery tool. It is the engine that powers the
[[GhostView]] desktop GUI for browsing Pacman, AUR/Chaotic-AUR, and Flatpak
packages. Part of the [[Ghost Ecosystem]]; by [[GhostKellz]].

Sits alongside the other Arch tooling: [[GhostView]] (GUI front-end), [[reaper]]
(AUR package manager), and [[ghostctl]] (which manages AUR-helper preferences).
