# GShell

A modern Linux shell designed to replace Bash/Zsh/Fish, using [[Ghostlang]]
(Lua-like syntax) for configuration and plugins. Written in [[Zig]]. Part of the
[[Ghost Ecosystem]]; by [[GhostKellz]].

## Notable

- **Ghostlang scripting** for config and plugins instead of POSIX shell script.
- **Powerful FFI** — 30+ shell functions exposed to scripts.

> [!key-insight]
> GShell is one half of the "Ghostlang everywhere" story: the same scripting
> engine configures both the shell and the [[Grim]] editor.

Built on [[Zig]].
