# ghostsnap

A fast, secure, deduplicating backup CLI for Linux. By [[GhostKellz]]; part of
the [[Ghost Ecosystem]].

Occupies the same data-protection space that [[ghostctl]] wraps via Restic/
Snapper/Btrfs workflows — a dedicated, restic-style deduplicating backup tool.
