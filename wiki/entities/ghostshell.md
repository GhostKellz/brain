# ghostshell

A fast, native, feature-rich terminal emulator that pushes modern features,
derived from Ghostty — usable as a native GUI or an embeddable library via
`libghostty`. By [[GhostKellz]]; part of the [[Ghost Ecosystem]].

> [!gap]
> This repo carries upstream Ghostty branding/text; treat it as a Ghostty-based
> terminal rather than a fully from-scratch project when citing it.

Distinct from [[GShell]] (the Ghostlang-scriptable shell) — ghostshell is the
terminal emulator, GShell is the shell that runs inside one.

Built on [[Zig]].
