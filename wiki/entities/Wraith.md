# Wraith

A modern, secure web server and reverse proxy built in [[Zig]] to replace nginx.
Designed for the zero-trust era with TOML configuration, broad protocol support,
and production-grade security integrations. By [[GhostKellz]]; part of the
[[Ghost Ecosystem]].

Fits the Zig networking stack alongside [[zquic]] (QUIC transport) and
[[zcrypto]] (crypto).

Built on [[Zig]].
