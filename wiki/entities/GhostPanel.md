# GhostPanel

"Portainer for Bolt" — a lightweight, intuitive web interface for managing
[[Bolt]] container environments. Just as Portainer simplified Docker management,
GhostPanel aims to do the same for the Bolt container platform: simple single-
container deployment and a familiar GUI for containers, images, volumes, and
networks. By [[GhostKellz]]; part of the [[Ghost Ecosystem]].

The management UI sibling to the [[Bolt]] runtime, alongside [[GhostForge]]
(gaming-focused Bolt front-end).
